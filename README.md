# B站网页功能实现

本项目实现了课程《用LangGraph搭建智能体|AI Agents in LangGraph》的代码实现
视频地址:https://www.bilibili.com/video/BV1bi421v7oD/?vd_source=58f2a7823ba15a4066fd1257f10d3320


## 技术栈

- **gradio**
  - 负责页面布局和交互功能。
- **tavily**
  - 搜索引擎api
- **sqlite**
  - 实现同步和异步持久化存储
